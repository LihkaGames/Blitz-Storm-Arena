# Blitz-Storm-Arena
A fast-paced 2D top-down shooter game for Android with campaign levels, survival mode, weapon upgrades, and modern dark-themed UI. Lightweight, addictive, and built for smooth mobile gameplay.

🚀 First official release of **Blitz Storm Arena** – a fast-paced 2D top-down shooter game for Android.

---

## ✨ Features

* 🔫 Multiple Weapons System (Pistol, Rifle, SMG)
* ⚔️ Campaign Mode with Levels
* ☠️ Survival Mode (Endless Gameplay)
* 🎯 Smooth Controls (Joystick + Aim & Fire)
* ⚡ Special Ability System
* 💰 In-game Currency & Upgrades
* 🏆 Awards & Ranking System

---

## 🎨 UI Highlights

* 🌑 Modern Dark Theme Design
* 💛 Yellow Accent System for Actions
* 🔵 Clean HUD with Real-Time Stats
* 📱 Optimized for Mobile Screens
